## parse

---
`(let parse (fun (_name) (...)))`
Parse a CSS colour. Can convert a name to a colour code, a 3 digits code to a 6 digits one (eg `#abc` to `#aabbcc`)


**Author**: [@SuperFola](https://github.com/SuperFola)

#### Parameter
- `_name`: String, colour name or colour code
